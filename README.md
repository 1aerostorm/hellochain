# hellochain

Same as hello world, but it is blockchain.

PoW. PoS. DPoS (planning).

Written in Rust.

## Requirements

Supposed for Linux.

Install Rust:  
https://www.rust-lang.org/tools/install

## Building 

```
cargo build
```

## Running

```
./target/debug/hellochain
```

It runs quick test of the blockchain.
